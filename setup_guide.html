Personal Ticket Assistant - Setup Guide

The application is currently running in Demo Mode with mock data. To use the application with your real Jira tickets, you need to configure your credentials.

⚠️ Configuration Required

Your application is running but needs Jira credentials to fully function.

Step 1: Get Your Jira API Token

  1. Go to https://id.atlassian.com/manage-profile/security/api-tokens
  2. Click "Create API token"
  3. Name it "Personal Ticket Assistant" and click "Create"
  4. Copy the generated token

Step 2: Update Your .env File

Edit the .env file in the root of your project and replace:

JIRA_API_TOKEN=your_jira_api_token_here

With your actual token:

JIRA_API_TOKEN=your-actual-token-here

Step 3: Restart the Application

After updating your .env file, stop the current process and restart the application:

./start_web.sh
